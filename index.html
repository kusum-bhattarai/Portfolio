Bobcat Bounty App

The Bobcat Bounty App is a comprehensive web application designed to streamline food distribution and inventory management for non-profit organizations. Built with TypeScript and React, it offers a user-friendly interface and robust functionality.

  • Key Features:
    • Secure user authentication and role-based access control.
    • Real-time inventory tracking and updates.

    • Analytics dashboard for visualizing resource allocation and usage trends.
  • Tech Stack: TypeScript, React, Node.js, PostgreSQL, Firebase (for cloud storage).
  • Purpose: To optimize food distribution processes and reduce waste for non-profits.
