Bobcat Bounty App

The Bobcat Bounty App is a comprehensive web application designed to streamline food distribution and inventory management for non-profit organizations. Built with TypeScript and React, it offers a user-friendly interface and robust functionality.

  • Key Features:
    • Secure user authentication and role-based access control.
    • Real-time inventory tracking and updates.
    • Event scheduling for food distribution with automated reminders.
    • Analytics dashboard for visualizing resource allocation and usage trends.
  • Tech Stack: TypeScript, React, Node.js, PostgreSQL, Firebase (for cloud storage).
  • Purpose: To optimize food distribution processes and reduce waste for non-profits.

Watched

Watched is a personal movie and TV show tracking application built with React and integrated with the TMDB API. It allows users to maintain a library of watched content and fetch additional details automatically.

  • Key Features:
    • Add, categorize, and rate movies and TV shows.
    • Fetch details like posters, summaries, and ratings from TMDB API.
    • Sort and filter library based on genres, ratings, and watch status.
    • Responsive design for seamless use on desktop and mobile.
  • Tech Stack: React, JavaScript, TMDB API, CSS3.
  • Purpose: To provide a personalized media tracking experience with rich data integration.
